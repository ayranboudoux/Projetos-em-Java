# Projetos em Java — Backend

Este repositório reúne meus projetos desenvolvidos em **Java**, desde os primeiros estudos até aplicações backend mais completas, com foco em **boas práticas** e **organização de código**.

O objetivo é documentar minha evolução como desenvolvedor backend e servir como **portfólio**.
